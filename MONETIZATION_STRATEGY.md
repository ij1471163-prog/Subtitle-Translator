## 💡 كيفية زيادة Conversion (FREE → PLUS)

### **1. Freemium Architecture (الحالي)**
```
30 دقيقة/يوم = 15 hours/week مجاني
└─ كافي للمستخدم العادي لكن بحد

عندما يصل الحد:
└─ عرض الترقية + dialog جميل
```

### **2. Timing الصحيح**
```
❌ عرض الترقية فوراً: 2% conversion
✅ بعد 3 أيام استخدام: 8% conversion
✅ عند الوصول للحد: 12% conversion
✅ عند فترة خمول 2 أسابيع: 15% conversion
```

### **3. Copy الترغيب**
```
❌ "ترقِّ الآن"
✅ "استمتع بترجمة غير محدود"

❌ "اشترك"
✅ "$0.99/شهر — توفير 33% على السنة"

❌ "المزيد من الميزات"
✅ "ترجم الآن بدون حدود — شاهد كل الفيديو"
```

### **4. Pain Points الاستخدام**
```
عند الوصول لـ 30 دقيقة:
├─ Show banner: "انتهت فترتك لليوم"
├─ Remaining time: واضح في الواجهة الرئيسية
├─ Dialog خفيف (no popup spam)
├─ Option: "غداً" بدون الضغط على شراء
└─ Goal: Build trust, not pushy
```

├─ Target: 5-8%
├─ Formula: PLUS users / FREE users

Retention Rate
├─ Day 1: 70% (Users who open app again)
├─ Day 7: 40%
├─ Day 30: 25%
└─ إذا أقل = App quality issue

Churn Rate (Unsubscribe)

├─ Formula: Cancelled subs / Active subs

## 💻 الكود (الحد الأدنى)

### **UserManager.java**
```java
public class UserManager {
    private SharedPreferences prefs;
    private static final long FREE_LIMIT_MS = 30 * 60 * 1000;
    private long translationStartTime;
    
    public boolean isSubscribed() {
        return prefs.getBoolean("is_subscribed", false);
    }
    
    public boolean canTranslate() {
        if (isSubscribed()) return true;
        
        String today = getToday();
        String lastDate = prefs.getString("last_date", "");
        
        if (!lastDate.equals(today)) {
            prefs.edit()
                .putLong("daily_usage", 0)
                .putString("last_date", today)
                .apply();
            return true;
        }
        
        long used = prefs.getLong("daily_usage", 0);
        return used < FREE_LIMIT_MS;
    }
    
    public void startTranslation() {
        translationStartTime = System.currentTimeMillis();
    }
    
    public void stopTranslation() {
        if (isSubscribed()) return;
        
        long duration = System.currentTimeMillis() - translationStartTime;
        long currentUsage = prefs.getLong("daily_usage", 0);
        
        prefs.edit()
            .putLong("daily_usage", currentUsage + duration)
            .apply();
    }
    
    public long getRemainingTime() {
        if (isSubscribed()) return Long.MAX_VALUE;
        
        long used = prefs.getLong("daily_usage", 0);
        return Math.max(0, FREE_LIMIT_MS - used);
    }
}
```
